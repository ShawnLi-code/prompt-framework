# 绿色版使用说明

## 特点

- **无安装**：解压即用，不写注册表，不创建服务
- **零 C 盘占用**：可以放在任何盘（推荐 F 盘）
- **不污染系统**：可以随时删除整个文件夹，卸载完成

## 使用方法

1. 把 `提示词框架.exe` 解压到任意位置（建议 `F:\ruanjian\`）
2. 双击运行
3. 第一次启动会自动打开设置窗口，填入你的 AI API 信息
4. 设置完成后关掉窗口，开始使用
5. 选中任意应用中的文字 → 按 `Ctrl+Alt+Shift+P` → 鼠标位置弹出小窗
6. 右键托盘图标可以退出 / 重新打开设置

## 启动后会发生什么

- 程序运行时会自动在 `托盘` 创建图标
- 你的 API Key、模板、配置保存在 `%APPDATA%\com.shawn.fanyi\settings.json`
  （这是 Windows 程序标准位置，没法避免，但只有几 KB）
- 程序通过 WebView2 渲染前端，第一次运行会创建 WebView2 缓存目录
  （位置 `%LOCALAPPDATA%\com.shawn.fanyi\`，几百 KB 到几 MB）

## 不需要的内容

- ❌ 不写 `C:\Program Files\`
- ❌ 不写注册表
- ❌ 不创建桌面快捷方式
- ❌ 不创建开始菜单项
- ❌ 不需要管理员权限

如果想要桌面快捷方式，可以右键 `提示词框架.exe` → 发送到 → 桌面快捷方式。
